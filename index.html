CLI Commands

Helium supports command-line arguments for scripting and automation:

Command Description
helium Interactive menu
helium --update Update blocklist database
helium --upgrade Update Helium script from repository
helium --status Show current status
helium --version Show version
helium --help Show help

Manual Cron Setup

Set up automatic daily updates manually using these commands:

Linux (Debian/Ubuntu/RHEL)

# Daily blocklist update at 4 AM echo "0 4 * * * root /usr/local/sbin/helium --update" | sudo tee -a /etc/crontab # Weekly script upgrade (optional) echo "0 5 * * 0 root /usr/local/sbin/helium --upgrade" | sudo tee -a /etc/crontab

OpenWrt

# Daily blocklist update at 4 AM echo "0 4 * * * /usr/sbin/helium --update" >> /etc/crontabs/root # Weekly script upgrade (optional) echo "0 5 * * 0 /usr/sbin/helium --upgrade" >> /etc/crontabs/root # Restart cron service /etc/init.d/cron restart
Note: Helium automatically sets up cron during installation. Use manual setup only if you need custom scheduling or the auto-setup didn't work.
